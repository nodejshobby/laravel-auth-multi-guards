## To run Laravel Auth System Locally

`composer install`

## copy content of .env.example to .env
`cp .env.example .env`

## Fill in your database configuration
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=
```
## Then start the development 

`php artisan serve`
